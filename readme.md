# When the Apple Doesn’t Fall Far from the Tree

This is a repository for the paper »When the Apple Doesn’t Fall Far from the Tree« by Pauline Maresi Kinzler, Pascale Willemsen, and me.

- `/analysis` contains data and a Jupyter Notebook (version 7.4.4) for analysis with Python (version 3.9.12).
- `/preregistration` contains materials from the preregistration.
- `/study` contains files from the study created with _Qualtrics_.
