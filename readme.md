# When the Apple Doesn’t Fall Far from the Tree

This is a repository for the paper »When the Apple Doesn’t Fall Far from the Tree« by Pauline Maresi Kinzler, Pascale Willemsen, and me.

- `/analysis` contains data and Jupyter Notebooks (version 7.4.4) for analysis with Python (version 3.9.12).
- `/pilot` contains files from pilot studies created with _Qualtrics_.
- `/preregistration` contains materials from preregistrations.
- `/study` contains files from studies created with _Qualtrics_.
